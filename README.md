# Royal Inn Lodge

This project is the website for Royal Inn Lodge, Shivajinagar, Bangalore.

## Getting Started

To run this project locally:

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open http://localhost:8080 to view it in the browser.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

This project is configured for deployment on Hostinger via Node.js integration.
The `start` command `vite preview --port 3000` is used for production serving.
